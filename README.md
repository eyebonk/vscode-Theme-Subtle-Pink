# Installation

## From the Marketplace (Recommended)

1. Install [Visual Studio Code](https://code.visualstudio.com/)
2. Launch Visual Studio Code
3. Open Extensions (`Cmd+Shift+X` on Mac / `Ctrl+Shift+X` on Windows)
4. Search for `subtle-pink`
5. Click **Install**
6. Open Command Palette (`Cmd+Shift+P` / `Ctrl+Shift+P`)
7. Type "Color Theme" and select **Preferences: Color Theme**
8. Choose **Subtle Pink**

## Manual Installation (VSIX)

For editors that don't have Marketplace access (e.g. Windsurf, Cursor, or other AI-powered editors):

1. Download the latest `.vsix` file from [GitHub Releases](https://github.com/eyebonk/vscode-Theme-Subtle-Pink/releases)
2. Open your editor
3. Open Command Palette (`Cmd+Shift+P` / `Ctrl+Shift+P`)
4. Type "Install from VSIX" and select **Extensions: Install from VSIX...**
5. Select the downloaded `.vsix` file
6. Open Command Palette and select **Preferences: Color Theme**
7. Choose **Subtle Pink**
